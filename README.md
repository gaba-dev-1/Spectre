## ADDITION 2 DIRECTIONS

<div align="center">

### Côte à Côte
```
N + E = ↑ + → = ↗
N + W = ↑ + ← = ↖  
S + E = ↓ + → = ↘
S + W = ↓ + ← = ↙
```

### Face à Face
```
N + S = ↑ + ↓ = ○ (s'annulent)
E + W = → + ← = ○ (s'annulent)
```

### Schéma Visuel
```
    ↖ N+W     N+E ↗
       ╲       ╱
        ╲  ↑  ╱
    ← ────○──── →
        ╱  ↓  ╲
       ╱       ╲
    ↙ S+W     S+E ↘
```

## ADDITION 3 DIRECTIONS

<div align="center">

### Schéma 3D
```
      N+E+W = N (↑)
         ╱│╲
        ╱ │ ╲
N+S+E=E  ○  N+S+W=W
        ╲ │ ╱
         ╲│╱
      S+E+W = S (↓)
```

### Résultats
```
N + E + W = ↑ (reste Nord)
N + S + E = → (reste Est)  
N + S + W = ← (reste Ouest)
S + E + W = ↓ (reste Sud)
```

```

        ○─────→ (4 km)
        │      │
    3km │      │
        │      │
        ↓      ●

Direction = 53° vers Nord-Est
```

```
5N + 2S + 3E + 6W

        2↓
```

## RÈGLES VISUELLES

<div align="center">

### Règle d'Annulation
```
↑ + ↓ = ○
← + → = ○

Opposés s'annulent toujours
```

### Règle Diagonale
```
↑ + → = ↗
↑ + ← = ↖
↓ + → = ↘
↓ + ← = ↙

Adjacents font des diagonales
```

### Règle Triple
```
3 directions = la 4ème manquante

↑ + → + ← = ↑
↑ + ↓ + → = →
```

## FORMULE SIMPLE

<div align="center">

### Addition Générale
```
aN + bS + cE + dW = Résultat

Vertical: (a-b) ↕
Horizontal: (c-d) ↔

Force finale = √[(a-b)² + (c-d)²]
```

### Exemple
```
2N + 1S + 3E + 1W

Vertical: 2-1 = 1 (↑)
Horizontal: 3-1 = 2 (→)

Résultat: 1↑ + 2→ = ↗
Force: √(1² + 2²) = 2.24
```
